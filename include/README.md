## ui.h

Reference Only

This `ui.h` defines the API surface supported by Uing. 
It serves as a reference for developers and AI tools to understand the API definitions, and is not for compilation.

Unlike the original `libui` or `libui-ng`, this file includes custom image display extensions by kojix2. 

https://github.com/kojix2/libui-ng/blob/dev/ui.h
